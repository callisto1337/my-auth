# my-auth
Небольшой пример работы авторизации на сайте. 

## Определяем каталог

```bash
$ cd myapp
```

## Устанавливаем зависимости

```bash
$ npm install
```

## Запускаем программу

```bash
$ node app.js
```

## Проверка

В браузере открываем адрес http://localhost:3000/. <br>
